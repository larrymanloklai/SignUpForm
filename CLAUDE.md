# SignUpForm Project

## Overview
A Norse/Viking-themed sign-up form built as an Odin Project exercise. Features a two-column layout with an Asgard background image on the left and a registration form on the right.

## File Structure
- `index.html` — Main HTML with the sign-up form (first name, last name, email, phone, password, confirm password)
- `styles.css` — Main stylesheet (two-column layout, form styling, error handling)
- `styles2.css` — Secondary stylesheet (unused or alternate)
- `javascript.js` — Password match validation on form submit
- `Asgard.webp` — Background image for left section
- `odin-lined.png` — Logo image
- `asgard-logo.jpeg`, `viking.png` — Additional assets
